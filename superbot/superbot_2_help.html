Controls:
play Start playing.
pp Toogle play/pause.
" next Play next song in the playlist.
stop Stop the playlist.
seek value|+/-value Seek to an absolute position (in secods). Use +value or -value to seek relative to the current position.

Volume:
v value - Set volume to value. If the value is omitted bot shows the current volume.
v+ Increase volume by 5% for each plus sign. For example #{cc}v+++++ increases the volume by 25%.
v- Decrease volume by 5% for each minus sign.

Channel control:
stick Sticks the bot to your current channel.
unstick unsticks the bot.
follow Let the bot follow you.
unfollow The bot stops following you.

Settings:
displayinfo Toggle where to show the current playling song; either in the comment or as a text message to the channel.
consume Toggle mpd´s consume mode which removes played titles from the playlist if on.
repeat Toogle mpd´s repeat mode.
random Toogle mpd´s random mode.
single Toogle mpd´s single mode.
crossfade seconds Set crossfade in seconds, set 0 to disable it.
ducking Toggle ducking mode.

Per user settings:
+ Hashtag add to Chatgroup
- Hashtag remove from Chatgroup
* show subscribed Chatgroups
following groups are aviable:
#volume, #update, #random, #single, #xfade, #repeat, #state

Playlists:
playlists Show a list of all playlists.
playlist number Load the playlist and start it. Use playlists to get a list of all playlists.
playlist Show all items of the currently loaded playlist + the name of it.
clear Clears the current queue.

Specials:
gotobed Let the bot mute and deaf himself and pause the playlist.
wakeup The opposite of gotobed.
ch Let the bot switch into your channel.
song Show the currently played song information.
If this information is empty, try file instead.
file Show the filename of the currently played song if song does not contain useful information.
help Shows this help.
stats Shows some MPD statistics.
